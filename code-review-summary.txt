[MEDIUM] GitHub Issue #197: Improper JavaScript module initialization
─────────────────────────────────────────────────────────────────────────────
File: js/index.js (lines 1-3)
Violation: JavaScript Guidelines - Minimal and modular code

Problem:
  The init() function uses dynamic import() inside a function, deferring 
  age-modal.js module loading until all HTMX partials are loaded. This pattern:
  - Creates potential race conditions with event listeners
  - Delays initialization of critical functionality  
  - Makes dependency loading unclear

Current Code:
  function init() {
    import("./age-modal.js");
  }

Recommendation:
  Either load age-modal.js as a static ES module at the top level, or 
  integrate its functionality directly into index.js with clear dependency 
  management.
