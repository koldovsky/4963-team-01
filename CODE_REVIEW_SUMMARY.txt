CRITICAL FINDINGS:
================================================================================

🔴 SCSS NESTED SYNTAX (Forbidden)
   
   The most critical violation is the use of SCSS nested syntax throughout
   the CSS files. This requires a build system/preprocessor, which AGENTS.md
   explicitly forbids:
   
   "Forbidden Patterns:
   - React, Vue, Angular
   - Tailwind
   - jQuery
   - CSS frameworks
   - build systems
   - bundlers"
   
   The project must remain a static website with plain HTML, CSS, and JS.
   
   Files Affected:
   • css/index.best-deals.partial.css (lines 10-116)
   • css/components.css (lines 37-50)
   • css/index.varieties.partial.css (nested selectors)
   • css/index.newsletter.partial.css (nested selectors)
   
   Action Required: Convert all nested rules to flat CSS using individual
   class selectors.

🟠 BEM NAMING VIOLATIONS

   Header navigation uses non-semantic abbreviation ".nv" instead of
   ".header__nav" pattern. This violates BEM methodology and AGENTS.md
   requirement for semantic class names.
   
   Current: .nv, .nv__container, .nv__menu, .nv__burger
   Required: .header__nav, .header__nav-container, etc.
